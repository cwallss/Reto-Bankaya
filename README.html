Reto Bankaya

Autor: Carlos Walls Salcedo

Fecha: Junio 2025

Objetivo del Proyecto

El objetivo principal de este reto es desarrollar un modelo de riesgo crediticio para Bankaya, capaz de clasificar a los clientes en buenos o malos pagadores. A partir de datos internos de solicitudes de préstamo y reportes externos de historial crediticio, se busca generar un riesgo_score que permita asignar tasas de interés dinámicas, mejorando así la toma de decisiones en la aprobación y pricing de créditos para la adquisición de smartphones.

Metodología

El proyecto se dividió en las siguientes etapas:

  • Exploración y limpieza de datos: Tratamiento de valores faltantes y negativos, codificación adecuada de variables.
  • Análisis exploratorio: Estudio de distribuciones, correlaciones y creación de una categoría interpretativa de riesgo.
  • Integración de historial crediticio externo: Limpieza, agregación e ingeniería de nuevas variables de riesgo por cliente.
  • Modelado predictivo: Entrenamiento de modelos (Regresión Logística, Árbol de Decisión, Random Forest, XGBoost, Red Neuronal). Se aplicó balanceo de clases con ROSE.
  • Asignación de tasas dinámicas: Con base en el score calculado por XGBoost, se asignaron tasas personalizadas entre el 10% y 50%.

Descripción de Datos

Se utilizaron dos conjuntos de datos principales:

  • main_dataset: Información de cada solicitud de préstamo, incluyendo fechas, atraso máximo, historial BNPL, consultas de crédito recientes, etc.
  • credit_reports: Historial financiero externo, con registros múltiples por cliente. Incluye tipo de crédito, uso del crédito, retrasos, saldos, entre otros.

Ambos datasets se integraron para enriquecer el modelado.

Conclusiones

  • XGBoost se seleccionó como el modelo final, con un Recall del 100% en la clase positiva (malos pagadores).
  • La variable max_days_late fue la más importante para la clasificación.
  • La tasa de interés se asigna de forma proporcional al riesgo individual estimado.
  • Este enfoque permite a Bankaya optimizar su cartera de préstamos e incentivar la inclusión financiera.
